# 快速开始

这份文档帮助你尽快把 `rn-alipay` 接入 React Native Android 项目。

## 一、适用范围

当前库：

- 只支持 Android
- 只负责客户端调起支付宝支付
- 要求服务端返回已签名的 `orderInfo`

## 二、安装

```bash
pnpm add rn-alipay
# 或 npm install rn-alipay
```

安装完成后，重新编译 Android：

```bash
npx react-native run-android
```

## 三、最小调用流程

### 1. 服务端创建订单

客户端先请求你的服务端：

```ts
const response = await fetch('https://your-api.example.com/pay/alipay/order', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    amount: '0.01',
    subject: '测试商品',
    outTradeNo: `order_${Date.now()}`,
  }),
});
```

### 2. 拿到 `orderInfo`

```ts
const { orderInfo } = await response.json();
```

### 3. 调起支付

```ts
import Alipay from 'rn-alipay';

const result = await Alipay.pay(orderInfo, { showLoading: true });
```

### 4. 正确处理返回结果

```ts
if (result.succeeded) {
  // 客户端同步回调成功
  // 最终仍以后端通知或查单结果为准
} else if (result.pending) {
  // 支付确认中
} else if (result.cancelled) {
  // 用户取消
} else {
  // 支付失败
}
```

## 四、联调前必须确认

- 宿主项目是 Android 真机环境
- 设备已安装支付宝客户端
- 服务端可以返回合法的 `orderInfo`
- 服务端已实现异步通知或订单查询
